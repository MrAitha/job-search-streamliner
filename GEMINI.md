# Job Search Streamliner Rules

## Candidate Profile
- **Target Roles:** IT Support, HRIS Analyst, Data Analyst.
- **Background:** IT training (CompTIA/Per Scholas), HR operations background.
- **Core Skills:** SQL, Power BI, Workday.
- **Experience Level:** Entry/Mid-level.

## Core Directive: Approval Chance Scoring
When evaluating job descriptions, the system must prioritize an "Approval Chance Score" (0-100) based strictly on alignment with the skills above. 
Jobs that heavily overlap with SQL, Power BI, Workday, and IT/HR operations should receive higher scores. Roles that require extensive senior experience (5+ years) or lack these core tools should be aggressively penalized in the scoring logic.
